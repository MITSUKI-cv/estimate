## two_choice_logit5
### 概要
- 3の反省を受け，段階推定から同時推定に変更
- これにより，各交通手段の影響を明らかにすることができる
- 推定結果は，resのものをスライドで紹介（20251216金沢大合同ゼミ）

### 推定結果
- 徒歩以外のモードのパラメータが有意でない（10%有意ですらない）
- 原因として考えられるのは，徒歩以外のサンプル数が少ないことがあげられる
- 個人的にはこの形がもっともシンプルで各モード別の違いを算出することができていると考える
- 有意でないパラメータにおいてはサンプルの標準誤差が大きいことが原因（推定誤差と推定値が同じくらいの値だとその値が0と変わらないと出るのは当たりまえ）

## DiPriTrip_MDCEV_estimate
### 概要
裁量私事活動のMDCEVモデルの作成
### モデル構造
- 目的変数
    - 最終トリップ後の帰宅時間（基準財）
    - 各タイミング
- 説明変数
    - 就業時間
    - 通勤メイン時間
    - 最寄端末時間（手段別の推定は一度あきらめ）
    - 世帯人数
    - 駅土地利用指標
※タイミングごとに各時間の違いを出すために共通変数＋偏差として効用関数を作成（タイミング2を基準としている）
